# -Los-Angeles-Crime-Data-Analysis
Comprehensive analysis of crime data in Los Angeles using Python, covering data cleaning, exploratory analysis, and visualization of 185,000+ crime records.
📊 Project Overview
Analysis of Los Angeles Police Department (LAPD) crime data with:

185,715 crime records
12 data features (crime type, location, victim info, date, weapon, etc.)
Geographic and temporal trend analysis

🔍 What's Included

Data Cleaning: Handle missing values, standardize formats
Exploratory Analysis: Crime trends, patterns, distributions
Visualizations: Crime hotspots, types, victim demographics, temporal trends

🛠️ Libraries Used

Pandas: Data manipulation & analysis
NumPy: Numerical computing
Matplotlib & Seaborn: Data visualization

📈 Key Analysis

Crime distribution by area/location
Crime types and frequency
Victim demographics (age, gender, descent)
Temporal patterns (dates, times, trends)
Weapons used in crimes
Status of investigations
Hotspot identification
